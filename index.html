บทคัดย่อ

คณะผู้จัดทำได้เล็งเห็นว่าการการสร้างเกมแนว Bingo 4 in a line ในภาษา C นั้นเป็นเรื่องที่น่าสนใจ จึงได้มีการศึกษาค้นคว้าวิธีการเขียนโค้ด

เเละการประยุกต์ลูกเล่นต่างๆให้เกมมีความสนใจ จึงได้เริ่มทำการศึกษาร่วมกัน โครงงานนี้เป็นการ Simulate Game Bingo 4 in A line

โดยมี Options ต่างๆเพื่อให้เกมสามารถ Function ได้หหลากหลาย เช่น Menu การใช้งาน, ระบบการเล่นแบบ Player vs Player หรือ Player vs AI,

วิธีการเล่นเกม, คำสั่งให้กดออก, การใส่ชื่อผู้เล่นและ คำสั่งกดเล่นอีกครั้ง พวกเราได้พัฒนาเป็นเกมที่เรีบยง่าย แต่ยังมีความน่าสนใจ และลูกเล่นต่างๆ

4.เพื่อศึกษาการทำงานของฟังก์ชั่นที่เกี่ยวข้อง

สรุป

หลังจากที่ได้พบอุปสรรคในการทํางานต่างๆ คณะผู้จัดทําได้วางเเผนการทํางานต่างๆเเบ่งกันศึกษาตามเเต่ละหัวข้อ

เพื่อร่วมมือกันเเก้ปัญหาโดยสามารถทําได้สําเร็จในภายหลัง โดยท้ายสุดเเล้วโปรเเกรมของพวกเราสามารถทํางานได้ครบทุกฟังก์ชั่น

ที่ได้ตั้งเป้าหมายไว้อย่างสมบูรณ์เเบบ สามารถ Display Menu การใช้งาน, ระบบการแข่งขันแบบ Player vs Player หรือ Player vs AI,

วิธีการเล่นเกม, คำสั่งให้กดออก, การใส่ชื่อผู้เล่นและ คำสั่งกดเล่นอีกครั้ง

ท้ายสุดคณะผู้จัดทําหวังว่าโครงงานนี้จะเป็นประโยชน์สําหรับผู้ที่สนใจใน C programming ในระดับเบื้องต้น
